docker

python ，pytorch，cuda都要相互兼容。可以用官方提供的docker，如nvidia。

cuda版本查看 ，其他的如nvidia-smi都不是。

docker run的时候要-v挂载数据， —shm-size扩大空间。docker工作目录命名为workspace，还得重新命名。

创建完容器之后重新打开

查看镜像 docker images

查看容器 docker ps

ssh

连接需要用户名、ip地址、密码。默认端口22。

可以在vscode里面下载ssh插件，打开服务器的文件夹

心得

docker也需要用 conda 来管理环境的，不过默认在用base。

vscode可直接连接到docker，再选择docker里面的conda python解释器，就可以跳转、查看参数了。

pylance类型检查报错，可以在vscode setting里面关闭。

最后能在服务器里面愉快的编程了。
